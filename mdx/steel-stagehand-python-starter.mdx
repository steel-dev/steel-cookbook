<Intro skipLink="#full-code-example">
Build intelligent browser automation with Steel's cloud browsers and Stagehand's AI-powered framework. Stagehand uses natural language instructions to interact with web pages, extract data with schemas, and perform complex actions autonomously.
</Intro>

<Step number={1} title="Install Required Packages">
Install the necessary dependencies. The Python SDK was rewritten in v3 and is published as `stagehand` (the old `stagehand-py` package is deprecated).

```bash
pip install steel-sdk stagehand python-dotenv
```
</Step>

<Step number={2} title="Set Up Environment Variables">
Create a `.env` file with your API keys.

```text filename=".env" /YOUR_STEEL_API_KEY/#1 /YOUR_OPENAI_API_KEY/#2
STEEL_API_KEY=YOUR_STEEL_API_KEY
OPENAI_API_KEY=YOUR_OPENAI_API_KEY
```
- Replace <HighlightRef id={2}>YOUR_OPENAI_API_KEY</HighlightRef> with your OpenAI API key
</Step>

<Step number={3} title="Define Extraction Schema">
Stagehand v3 uses JSON Schema (dicts) for structured extraction — no Pydantic/Zod.

```python showLineNumbers filename="main.py" /YOUR_STEEL_API_KEY/#1 /YOUR_OPENAI_API_KEY/#2
import asyncio
import os
from dotenv import load_dotenv
from steel import Steel
from stagehand import AsyncStagehand

# Load environment variables
load_dotenv()

# Replace with your own API keys
STEEL_API_KEY = os.getenv("STEEL_API_KEY") or "YOUR_STEEL_API_KEY"
OPENAI_API_KEY = os.getenv("OPENAI_API_KEY") or "YOUR_OPENAI_API_KEY"

STORY_SCHEMA = {
    "type": "object",
    "properties": {
        "stories": {
            "type": "array",
            "items": {
                "type": "object",
                "properties": {
                    "title": {"type": "string"},
                    "rank": {"type": "integer"},
                },
                "required": ["title", "rank"],
            },
        }
    },
    "required": ["stories"],
}
```
- Replace <HighlightRef id={2}>YOUR_OPENAI_API_KEY</HighlightRef> with your actual OpenAI API key
</Step>

<Step number={4} title="Create Steel Session">
Initialize a Steel browser session for Stagehand to connect to.

```python showLineNumbers{32} filename="main.py"
async def main():
    print("🚀 Steel + Stagehand Python Starter")
    print("=" * 60)

    if STEEL_API_KEY == "YOUR_STEEL_API_KEY":
        print("⚠️  WARNING: Please replace 'YOUR_STEEL_API_KEY' with your actual Steel API key")
        return

    if OPENAI_API_KEY == "YOUR_OPENAI_API_KEY":
        print("⚠️  WARNING: Please replace 'YOUR_OPENAI_API_KEY' with your actual OpenAI API key")
        return

    session = None
    session_id = None
    stagehand = None
    client = None

    try:
        client = Steel(steel_api_key=STEEL_API_KEY)
        session = client.sessions.create()

        print(f"Steel Session created!")
        print(f"View session at {session.session_viewer_url}")

        cdp_url = f"{session.websocket_url}&apiKey={STEEL_API_KEY}"
```
</Step>

<Step number={5} title="Configure Stagehand v3">
Stagehand v3 runs an embedded local server. To drive a Steel-hosted browser, pass `browser={"type": "local", "launchOptions": {"cdpUrl": ...}}` to `sessions.start`:

```python showLineNumbers{58} filename="main.py"
        stagehand = AsyncStagehand(
            server="local",
            model_api_key=OPENAI_API_KEY,
            local_ready_timeout_s=30.0,
        )

        stagehand_session = await stagehand.sessions.start(
            model_name="openai/gpt-5",
            browser={
                "type": "local",
                "launchOptions": {"cdpUrl": cdp_url},
            },
        )
        session_id = stagehand_session.data.session_id

        print("Connected to browser via Stagehand")
```
</Step>

<Step number={6} title="Navigate and Extract Data">
v3 AI operations stream Server-Sent Events; drain the stream to get the final result.

```python showLineNumbers{74} filename="main.py"
        async def stream_to_result(stream, label):
            result = None
            async for event in stream:
                if event.type == "log":
                    continue
                if event.data.status == "finished":
                    result = event.data.result
                elif event.data.status == "error":
                    raise RuntimeError(event.data.error or "unknown error")
            return result

        await stagehand.sessions.navigate(id=session_id, url="https://news.ycombinator.com")

        extract_stream = stagehand.sessions.extract(
            id=session_id,
            instruction="Extract the titles and ranks of the first 5 stories on the page",
            schema=STORY_SCHEMA,
            stream_response=True,
            x_stream_response="true",
        )
        stories_data = await stream_to_result(extract_stream, "extract")

        print("\nTop 5 Hacker News Stories:")
        for story in (stories_data or {}).get("stories", []):
            print(f"{story['rank']}. {story['title']}")
```
</Step>

<Step number={7} title="Perform AI Actions">
Use `act` for natural-language interactions. Pick an action you know exists on the page so the LLM doesn't have to guess.

```python showLineNumbers{100} filename="main.py"
        try:
            act_stream = stagehand.sessions.act(
                id=session_id,
                instruction="click the 'new' link in the top navigation",
                stream_response=True,
                x_stream_response="true",
            )
            await stream_to_result(act_stream, "act")
            print("Navigated to new stories!")

        except Exception as error:
            print(f"Could not navigate to new stories: {error}")

        print("\nAutomation completed successfully!")
```
</Step>

<Step number={8} title="Clean Up Resources">
Always end the Stagehand session, close the client, and release the Steel session.


<FullCodeExample id="full-code-example" title="Full Example">
Complete AI-powered browser automation using Stagehand with Steel.
```python showLineNumbers filename="main.py"
"""
AI-powered browser automation using Stagehand with Steel browsers.
"""

import asyncio
import os
import sys
from dotenv import load_dotenv
from steel import Steel
from stagehand import AsyncStagehand

load_dotenv()

STEEL_API_KEY = os.getenv("STEEL_API_KEY") or "your-steel-api-key-here"
OPENAI_API_KEY = os.getenv("OPENAI_API_KEY") or "your-openai-api-key-here"

STORY_SCHEMA = {
    "type": "object",
    "properties": {
        "stories": {
            "type": "array",
            "items": {
                "type": "object",
                "properties": {
                    "title": {"type": "string"},
                    "rank": {"type": "integer"},
                },
                "required": ["title", "rank"],
            },
        }
    },
    "required": ["stories"],
}


async def stream_to_result(stream, label):
    result = None
    async for event in stream:
        if event.type == "log":
            continue
        if event.data.status == "finished":
            result = event.data.result
        elif event.data.status == "error":
            raise RuntimeError(event.data.error or "unknown error")
    return result


async def main():
    print("🚀 Steel + Stagehand Python Starter")
    print("=" * 60)

    if STEEL_API_KEY == "your-steel-api-key-here":
        print("⚠️  WARNING: Please set STEEL_API_KEY")
        sys.exit(1)

    if OPENAI_API_KEY == "your-openai-api-key-here":
        print("⚠️  WARNING: Please set OPENAI_API_KEY")
        sys.exit(1)

    session = None
    session_id = None
    stagehand = None
    client = None

    try:
        client = Steel(steel_api_key=STEEL_API_KEY)
        session = client.sessions.create()
        print(f"View session at {session.session_viewer_url}")

        cdp_url = f"{session.websocket_url}&apiKey={STEEL_API_KEY}"

        stagehand = AsyncStagehand(
            server="local",
            model_api_key=OPENAI_API_KEY,
            local_ready_timeout_s=30.0,
        )

        stagehand_session = await stagehand.sessions.start(
            model_name="openai/gpt-5",
            browser={"type": "local", "launchOptions": {"cdpUrl": cdp_url}},
        )
        session_id = stagehand_session.data.session_id

        await stagehand.sessions.navigate(id=session_id, url="https://news.ycombinator.com")

        extract_stream = stagehand.sessions.extract(
            id=session_id,
            instruction="Extract the titles and ranks of the first 5 stories on the page",
            schema=STORY_SCHEMA,
            stream_response=True,
            x_stream_response="true",
        )
        stories_data = await stream_to_result(extract_stream, "extract")

        print("\nTop 5 Hacker News Stories:")
        for story in (stories_data or {}).get("stories", []):
            print(f"{story['rank']}. {story['title']}")

    except Exception as error:
        print(f"Error during automation: {error}")
        raise

    finally:
        if stagehand and session_id:
            try:
                await stagehand.sessions.end(id=session_id)
            except Exception:
                pass
        if stagehand:
            await stagehand.close()
        if session and client:
            client.sessions.release(session.id)


if __name__ == "__main__":
    asyncio.run(main())
```
