# Custom Skills

Place custom skills and agent instructions in this directory.

Each skill file will be discovered and loaded by your favourite AI coding agent (Claude Code, Codex, Amp code, OpenCode, Crush, etc.) alongside the project's AGENTS.md.

Example skills you might add:
- Testing strategies specific to this example
- Code style preferences
- Framework-specific conventions
- Deployment workflows
